Your Goals!

  • If someone tries to high five themself... stop them
  • Allow for some random enthusiasm when a creature is created
  • Allow for a successful high five to provide different levels of enthusiasm
  • Allow a creature to have one doubleTap, which doubles the returned enthusiasm. If it is a miss, the creature is removed from the game.
  • Bonus: On page load, give the user an option to select who they want to play with from a list of creatures
